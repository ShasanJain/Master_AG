# Economics Writing

When writing economic content:
1. **Bottom Line Up Front (BLUF):** State the core economic impact in the first paragraph.
2. **Causality vs Correlation:** Be hyper-specific about causal mechanisms. Use "X caused Y because Z" rather than "X is linked to Y".
3. **Quantify:** "Inflation rose 2.1%" not "Inflation saw a significant increase".
4. **Anti-Slop:** Remove "dynamic landscape", "robust economy", and "pivotal shifts". Use standard economic terminology instead of flowery metaphors.
