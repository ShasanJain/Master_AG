# Scientific Writing

When writing scientific content:
1. **Precision First:** Avoid vague descriptors. Use exact measurements, p-values, and defined variables.
2. **Anti-Slop:** Do not use "delve", "testament", or "tapestry". Strip "Challenges and Future Outlook" padding.
3. **Data Over Narrative:** Let the data speak. Do not over-claim significance.
4. **Reproducibility:** Clearly state methods, hyperparameters, and environment configurations.
