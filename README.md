# Master-AG 🤖

Master-AG is a specialized, agent-driven workspace operating on a highly robust 3-layer orchestration architecture. This repository is built to safely accelerate autonomous workflows, enforce rigid testing constraints, and natively declutter AI execution logic.

## 🚀 The 3-Layer Architecture

1. **Layer 1: Directives (Intent)**
   Standard Operating Procedures (SOPs) outlined in `AGENTS.md` dictating the core behavior loop. This includes a strict "Validation Gate" that physically prevents the system from completing tasks without natively passing zero-error tests first.

2. **Layer 2: Orchestration (Decision-Making)**
   The cognitive AI layer that dynamically routes intent to execution, ensuring code logic and architectural decisions perfectly match the directives without explicit human hand-holding.

3. **Layer 3: Execution (Doing The Work)**
   A library of over 100 standardized, deep-level meta-skills inside the `/skills` folder (e.g., `theme_factory_skill.md`, `test_driven_development_skill.md`). These modular tools strictly follow a fluid `_[name]_skill.md` naming architecture.

## 🛠️ System Integrity
This workspace is designed to self-heal. Its structural integrity is continually maintained by automated `organize-skills` scripts to gracefully handle missing files, auto-format configurations, and completely prevent workspace rot over time.
